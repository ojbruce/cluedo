\subsection{\-Description détaillée}
\hyperlink{classPiece}{\-Piece} est la classe représentant les cases d'une piece. 

\-Une \hyperlink{classPiece}{\-Piece} est caractérisé par les informations suivantes \-: un nom un pointeur vers la porte de la case (pas pris en charge)
